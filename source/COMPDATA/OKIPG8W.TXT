The following Oki printer, which might have been installed on your system, will not function properly under Microsoft Windows XP using the printer driver that is currently installed:

Okipage 8w Hi-per W

To obtain a more recent version of this printer driver, visit the manufacturer's Web site at http://www.microsoft.com/isapi/redir.dll?prd=whistler&ar=Printing&O1=oki. Web addresses can change, so you might be unable to connect to this Web site.
